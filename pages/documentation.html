Начало работы

Добро пожаловать в документацию BPM Centr! Здесь вы найдете всю необходимую информацию для начала работы с нашими коннекторами для платформы Make.

Шаг 1: Регистрация

Для начала работы с BPM Centr необходимо зарегистрироваться на платформе. Процесс регистрации занимает всего несколько минут.

Шаг 2: Выбор коннектора

После регистрации вы можете выбрать нужный коннектор из каталога коннекторов. Каждый коннектор имеет подробное описание функциональности и примеры использования.

Шаг 3: Оформление подписки

Выбрав нужный коннектор, вы можете оформить подписку на него. Каждый коннектор имеет бесплатный 7-дневный пробный период.

Шаг 4: Использование коннектора

После оформления подписки вы получите API-ключ, который необходимо использовать для подключения коннектора к вашему аккаунту Make.

Коннекторы

BPM Centr предлагает широкий выбор коннекторов для интеграции с различными системами и сервисами.

Категории коннекторов

  • Стандартные коннекторы (15 USD / месяц) - маркетплейсы, почтовые сервисы, SMS-сервисы
  • Расширенные коннекторы (25 USD / месяц) - платежные системы, сервисы аналитики
  • Премиум коннекторы (40 USD / месяц) - CRM-системы, ERP-системы, корпоративные решения

Модель подписки

Каждый коннектор оплачивается отдельно и может использоваться в одном аккаунте Make. Это означает, что вы можете использовать коннектор в любом количестве сценариев внутри одного аккаунта Make.

Пробный период

Каждый коннектор имеет бесплатный 7-дневный пробный период, в течение которого вы можете ознакомиться с его функциональностью без ограничений.

API

BPM Centr предоставляет API для управления подписками и коннекторами программным способом.

Аутентификация

Для аутентификации в API используется API-ключ, который можно получить в личном кабинете.

Примеры запросов


// Получение списка активных коннекторов
GET /api/v1/connectors
Authorization: Bearer YOUR_API_KEY

// Получение информации о подписке
GET /api/v1/subscriptions/{subscription_id}
Authorization: Bearer YOUR_API_KEY

Интеграция с Make

Коннекторы BPM Centr легко интегрируются с платформой Make.

Подключение коннектора

  1. Войдите в свой аккаунт Make
  2. Создайте новый сценарий или откройте существующий
  3. Добавьте модуль и выберите коннектор BPM Centr
  4. Введите API-ключ, полученный в личном кабинете BPM Centr
  5. Настройте параметры коннектора в соответствии с вашими потребностями

Примеры сценариев

Примеры сценариев для различных коннекторов доступны в разделе документации каждого коннектора.

Часто задаваемые вопросы

Как работает модель "один коннектор = один аккаунт Make"?

Каждая подписка на коннектор дает право использовать его в одном аккаунте Make. Коннектор привязывается к аккаунту Make при первом использовании и может использоваться в любом количестве сценариев внутри этого аккаунта.

Могу ли я использовать один коннектор в нескольких аккаунтах Make?

Нет, каждая подписка на коннектор привязывается к одному аккаунту Make. Если вам нужно использовать коннектор в нескольких аккаунтах, необходимо оформить отдельную подписку для каждого аккаунта.

Как происходит оплата?

Оплата производится ежемесячно или ежегодно (со скидкой 15%) с помощью банковской карты или банковского перевода. Вы можете управлять своими платежными данными в личном кабинете.

Есть ли ограничения по количеству операций?

Нет, подписка не ограничивает количество операций, которые можно выполнить с помощью коннектора. Вы можете использовать коннектор без ограничений в рамках одного аккаунта Make.
