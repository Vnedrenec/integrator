Описание

Creatio Коннектор предоставляет интеграцию с CRM-системой Creatio для автоматизации работы с контактами, сделками, задачами и другими объектами. Коннектор позволяет создавать, обновлять, удалять и получать данные из Creatio, а также реагировать на события в системе.

Ключевые особенности

  • Полная интеграция с Creatio CRM
  • Работа с контактами, компаниями, сделками и задачами
  • Автоматическое обновление данных
  • Триггеры для реагирования на события в Creatio
  • Поддержка пользовательских полей и объектов

Для кого предназначен

Коннектор предназначен для компаний и специалистов, использующих Creatio CRM и желающих автоматизировать бизнес-процессы с помощью платформы Make. Особенно полезен для:

  • Маркетологов, которым нужно автоматизировать работу с лидами
  • Менеджеров по продажам, которым нужно автоматизировать работу со сделками
  • Руководителей проектов, которым нужно автоматизировать работу с задачами
  • Интеграторов, разрабатывающих комплексные решения на базе Creatio

Совместимость

Коннектор совместим со следующими версиями Creatio:

  • Creatio 7.x и выше
  • Creatio Sales
  • Creatio Marketing
  • Creatio Service
  • Creatio Studio

Возможности

Модуль "Контакты"

Операции для работы с контактами в Creatio:

  • Получить контакт - получение информации о контакте по ID или другим параметрам
  • Создать контакт - создание нового контакта в системе
  • Обновить контакт - обновление информации о существующем контакте
  • Удалить контакт - удаление контакта из системы
  • Поиск контактов - поиск контактов по различным параметрам

Модуль "Компании"

Операции для работы с компаниями в Creatio:

  • Получить компанию - получение информации о компании по ID или другим параметрам
  • Создать компанию - создание новой компании в системе
  • Обновить компанию - обновление информации о существующей компании
  • Удалить компанию - удаление компании из системы
  • Поиск компаний - поиск компаний по различным параметрам

Модуль "Сделки"

Операции для работы со сделками в Creatio:

  • Получить сделку - получение информации о сделке по ID или другим параметрам
  • Создать сделку - создание новой сделки в системе
  • Обновить сделку - обновление информации о существующей сделке
  • Удалить сделку - удаление сделки из системы
  • Поиск сделок - поиск сделок по различным параметрам

Триггеры

Триггеры для реагирования на события в Creatio:

  • Новый контакт - срабатывает при создании нового контакта
  • Обновление контакта - срабатывает при обновлении контакта
  • Новая компания - срабатывает при создании новой компании
  • Обновление компании - срабатывает при обновлении компании
  • Новая сделка - срабатывает при создании новой сделки
  • Обновление сделки - срабатывает при обновлении сделки
  • Изменение стадии сделки - срабатывает при изменении стадии сделки

Примеры использования

Вот несколько примеров использования коннектора в сценариях Make:

  1. Автоматическое создание задач - при создании новой сделки в Creatio автоматически создавать задачу для менеджера
  2. Синхронизация контактов - синхронизация контактов между Creatio и другими системами
  3. Уведомления о сделках - отправка уведомлений в мессенджеры при изменении стадии сделки

Часто задаваемые вопросы

Как подключить Creatio Коннектор к моему аккаунту Make?

Для подключения коннектора необходимо приобрести подписку, получить API-ключ и добавить его в настройках коннектора в Make.

Какие версии Creatio поддерживаются?

Коннектор поддерживает Creatio версии 7.x и выше, включая Creatio Sales, Marketing, Service и Studio.

Как получить техническую поддержку?

Техническая поддержка доступна через тикет-систему в личном кабинете BPM Centr. Время ответа зависит от вашего тарифного плана.
